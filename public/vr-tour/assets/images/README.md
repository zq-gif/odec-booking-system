# 360 Images Folder

Place your 360-degree panoramic images here.

## Required Images

- `room1.jpg` - First location/room
- `room2.jpg` - Second location/room
- `room3.jpg` - Third location/room
- `arrow.png` - Navigation arrow icon (optional)

## Image Requirements

- **Format**: JPG, PNG
- **Projection**: Equirectangular (360 panorama)
- **Recommended Resolution**: 4096 x 2048 pixels or higher
- **Aspect Ratio**: 2:1

## Where to Get 360 Images

### Free Stock 360 Images:
- [Flickr Equirectangular Group](https://www.flickr.com/groups/equirectangular/)
- [Poly Haven HDRIs](https://polyhaven.com/hdris) (can be used as 360 backgrounds)
- [360 Cities](https://www.360cities.net/)

### Create Your Own:
- Use a 360 camera (Ricoh Theta, Insta360, GoPro Max, etc.)
- Use a smartphone with 360 camera app (Google Street View app, etc.)
- Stitch photos together using panorama software

## Optimization Tips

- Compress images to reduce file size (use tools like TinyPNG or ImageOptim)
- Balance quality vs loading time
- Consider progressive JPEGs for better loading experience
